# Project OneWay

OneWay 是一个翻墙工具包，用于简化和复用其它翻墙工具，加速二次开发, 可视为 V2Ray 的 Python 版本。

## 主要特点
* 多对多服务器支持，负载均衡
* 支持多用户
* 开放协议支持，兼容 ShadowSocks 和 GoAgent
